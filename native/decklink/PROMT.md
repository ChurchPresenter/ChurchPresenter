The DeckLink JNI native library needs to be rebuilt. On Windows use `native/decklink/build_now.bat` (not the CMake flow — CMake regenerates headers that lack legacy device support). On macOS use `native/decklink/build_macos.sh`. On Linux use `native/decklink/build_linux.sh`. See `native/decklink/BUILD.md` for full instructions. After building, copy the output binary to the matching resource directories under `composeApp/src/jvmMain/resources/` and `composeApp/src/jvmMain/appResources/`.
